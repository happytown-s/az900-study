# AZ-900 Study

Microsoft Azure Fundamentals (AZ-900) Practice Application

## Features

- 300 practice questions covering all AZ-900 exam objectives
- 10 categories with 30 questions each
- Three quiz modes: Drill (10 Qs), Exam (40 Qs), Review (20 Qs)
- EN/JA language support
- Progress tracking with category breakdown
- Estimated AZ-900 score (passing: 700/1000)
- Hint system and detailed explanations

## Categories

1. Cloud Concepts
2. Azure Core Services
3. Azure Compute
4. Azure Networking
5. Azure Storage
6. Azure Identity & Security
7. Azure Governance
8. Azure Cost Management
9. Azure Monitoring
10. Azure Solutions

## Tech Stack

- React + TypeScript + Vite
- Tailwind CSS 4
- LocalStorage for progress persistence

## Setup

```bash
npm install
npm run build
```
